About me

Turning ideas into real,
working software

I'm Sujal Wagholikar — a self-taught full-stack developer who cleared Class 12 while building production-grade web applications used by real users across India. No bootcamp, no shortcuts, just a laptop and relentless curiosity.

I specialize in building tools that actually solve problems — from PDF editors with OCR, to a national-scale civic voting portal, to exam simulators for English learners. Every line of code I write is meant to ship, not just to demo.

Currently focused on full-stack JavaScript (React + Node), cloud deployments via Cloudflare Workers, and building scalable backend systems with MongoDB. Always open to freelance projects and collaborations.
